How do I install and configure

Installation of #B4mad Racing Copilots is easy:

  1. Install CrewChief on your gaming rig
  2. configure CrewChief for your user

Install CrewChief

Install CrewChief in your gaming PC. Download the CrewChiefV4.msi installer via the forum.

{# djlint:off #}

Configure CrewChief

  1. Enable the telemetry collection by checking 'MQTT Telemetry enabled' in the CrewChief Properties
  2. Choose a 'MQTT drivername' in the CrewChief Properties cc_settings_mqtt, this must be the same as the one on your #B4mad Racing profile
  3. Enable text to speech
If all works you should see connection messages in the CrewChief

{# djlint:on #}
